Detailed Description

creates solver class using RGF-GPU for factorising, solving and selectively inverting linear system.

divided into set up, symbolic factorisation, numerical factorisation, numerical factorisation & solve and selected inversion (of the diagonal elements)

Note
in each RGFSolver function call factorise, selected_inversion etc. class RGF gets created. Is this the best way to handle things. Potentially merge them somehow? Maybe actually does not take any time.
